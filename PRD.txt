Assignment 1 (90 mintus):
Build a simple 2D plan viewer using React (or another frontend framework of your choice).
Frontend only (no backend required).
Functional Requirements
Viewer
● Display a provided plan image (attached below) , inside a viewer area.
● Zoom in/out using mouse wheel / trackpad scroll:
○ Zoom must be centered on the cursor position.
○ Apply reasonable limits (e.g., min 0.25x, max 5x).
● Pan the image using click-and-drag:
○ Mouse down + drag moves the image.
○ Releasing the mouse stops panning.
● Double click resets the view by centering the image in the viewer.
Action Log
● Display an action log table next to the viewer (side-by-side layout).
● The table must log user interactions in real time, including:
○ Zoom in / zoom out
○ Pan (start + end, or a single “pan” entry with delta)
○ Center reset (double click)
For each log entry include at minimum:
● Timestamp
● Action type
● Bonus task - Details (e.g., zoom level, pan delta, center/reset)
