# 🏗️ 2D Plan Viewer - Spacial.io

A modern, interactive 2D architectural plan viewer built with React + TypeScript + Vite.

## ✨ Features

- 🔍 **Cursor-Centered Zoom** — Scroll to zoom in/out, centered on your mouse position (0.25x–5x)
- ↔️ **Click & Drag Pan** — Mouse down + drag to navigate the plan
- 🔄 **Double-Click Reset** — Instantly re-center and fit the plan to view
- 📊 **Real-Time Action Log** — Every interaction logged with timestamp and details
- 🌓 **Light/Dark Mode** — Toggle between themes with persistence
- 📱 **Responsive Design** — Works on desktop, tablet, and mobile
- 🎨 **Modern Blue/White UI** — Gradient accents, smooth hover effects, emoji throughout

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
